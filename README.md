# Legal Beagle Skills

This is a collection of speciliazed Claude Skills designed to equip Claude with edge legal expertise. 

This repository provides workflwos, prompts and scripts for case analysis, legal research, document drafting, inspired by the daily tasks handled by legal professionals. 

Legal Beagle Skills turn Claude from a general-purpose AI into a specialized legal agent with procedural knowledge of law and legal operations.

## Getting Started

Using Skills in Claude.ai - On the Web.

Navigate to Settings > Capabilities

Enable the Skills toggle if off. 

Upload the Skill.md file and run the skill.
